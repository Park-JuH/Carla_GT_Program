# Carla_GT_Program

## 프로젝트 소개
Carla에서 추출한 데이터를 이용하여 Ground Truth를 생성하는 프로그램 개발

## 개발 기간
2022.09. - 2023.02

## 맴버 구성
박재준: Carla에서 GT를 만들기 위해 필요한 데이터 추출<br/>
박주혁: 추출한 데이터를 기반으로 GT 생성

## 개발 환경
- python: 3.71

## 최종 출력물
- RGB Image
- Json
